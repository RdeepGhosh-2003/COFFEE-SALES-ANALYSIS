## üß∞ SQL-Based Data Transformation

- Renamed and cleaned column headers
- Created queries for:
  - Monthly summaries
  - Daily metrics
  - Category-level and time-based breakdowns
  - Top-selling product lists

> SQL queries are documented in the `COFFEE-SALES.pptx` file.
